# AI Usage Disclosure

This repository was developed with substantial AI assistance as part of a final project for CSCI 4052U (Machine Learning 2). This disclosure is included to support transparent documentation of how AI tools were used during the project.

## Tools Used

- OpenAI Codex gpt-5.4
- Anthropic Claude Sonnet & Opus 4.5
- Google Gemini 3.1 Pro

## How AI Was Used

AI tools were used throughout the project lifecycle, including:

- brainstorming project direction and implementation options
- generating and revising project plans from high-level ideas
- drafting and modifying code
- refactoring existing code
- drafting and revising documentation
- generating and expanding tests
- discussing design decisions and tradeoffs
- generating commit messages
- producing research-style reports about repository progress and project state
- reviewing code and surfacing potential issues during development

## Human Role And Responsibility

The project remained under my supervision throughout development. My responsibilities included:

- setting project direction and goals
- providing context, constraints, and prompt instructions to the AI tools
- deciding which proposed changes to accept or reject
- play testing and otherwise evaluating outputs
- deciding when to pivot approaches based on intermediate results

AI tools acted as collaborators under supervision, not as autonomous owners of the project. I take final responsibility for the contents of this repository and for the final submitted work.

## Working Process

In practice, I often used Codex and Claude to turn an idea into an execution plan, then iterated on that plan by supplying additional context, monitoring progress, and redirecting work when needed. I also used AI-generated research reports and code review feedback to inspect the repository at different stages, track progress, and identify follow-up work.

## Notes On Interpretation

Because AI assistance was used extensively, this repository should be understood as an AI-assisted project rather than a project written entirely by hand. The purpose of this disclosure is transparency, not to minimize the human role in directing, evaluating, and submitting the work.

This statement is intended to satisfy course transparency expectations for AI use while accurately describing the development process behind this repository.
